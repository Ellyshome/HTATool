# 住宅景观水系净水技术手册

## 前言

本手册面向住宅景观设计、施工与运维团队，系统介绍景观水体的水质分类、常见污染机理、净化技术体系及关键量化参数，旨在为住宅景观水系的规划设计、施工实现与长期维护提供标准化参考。

---

# **第一部分：基本概念篇**

## 1. 水质分类

### 1.1 按功能用途分类

* **观赏级水体标准（景观用水）**
  用于视觉、环境营造，不直接接触人体，要求水质不低于Ⅳ级标准。


* **戏水级水体标准（可接触）**
  存在身体接触，如儿童戏水区域、浅水涉水带，不低于Ⅲ级标准。对细菌总数、浊度、藻类更严格。

## 2. 污染物的分类与成因

### 2.1 物理性污染

* **悬浮颗粒**：泥土、尘埃、落叶碎屑。
* **浑浊与沉积物**：景观施工残留、雨水带入。

### 2.2 化学性污染

* **营养盐**：氮、磷，来自：

  * 雨水径流中的肥料残留
  * 土壤溶出
  * 枯枝落叶分解
* **金属类（极少见）**：来自特殊水源或金属构件腐蚀。

### 2.3 生物性污染

* **藻类繁殖（绿水、蓝藻）**
  原因：高氮磷、光照过强、水体静止。

* **水华（藻类大量繁殖）**
  表现：绿色、黄绿色、蓝绿色表层漂浮层。

* **生物膜（slime）**
  原因：水体富营养 + 循环不足。

### 2.4 常见水质问题的表象与成因

| 表现       | 成因              |
| -------- | --------------- |
| 水体发绿     | 绿藻繁殖、营养盐过高      |
| 蓝绿色漂浮物   | 蓝藻爆发、静水、曝气不足    |
| 黑臭       | 底泥厌氧、腐败、循环差     |
| 水面油膜     | 花粉、植物油脂、水生动物分泌物 |
| 红水 / 茶色水 | 铁锈或腐叶多酚溶出       |

# **第二部分：水质净化措施篇**

## 1. 物理净化类 措施

### 1.1 机械过滤

* **表面撇渣（Skimmer）**：去除漂浮叶片、油膜。
* **重力砂滤器**：适合观赏水与戏水级净化。
* **纤维束过滤**：适用于高浊度水体的强化处理。

### 1.2 凝絮 + 沉淀

* 常用药剂：聚合氯化铝（PAC）、聚丙烯酰胺（PAM）。
* 适用于：

  * 初期浑浊水体治理
  * 富营养化初期控制

### 1.3 曝气

* 增加溶解氧，促进有益菌群活性。
* 类型：喷泉曝气、底曝、微孔曝气盘。

---

## 2. 生物净化（生态法）

### 2.1 水生植物净化

**沉水植物**：金鱼藻、狐尾藻、苦草

* 功能：吸收氮磷、与藻类竞争营养、释放抑藻物质。

**挺水植物**：芦苇、香蒲、黄菖蒲

* 功能：根区硝化/反硝化系统形成天然生物滤床。

### 2.2 水生动物净化

* **鲢、鳙（滤食性鱼类）**：滤食藻类与浮游生物。
* **贝类（蚌、三角帆蚌）**：高效滤食悬浮颗粒。

### 2.3 微生物法（生物膜、硝化体系）

* 生物滤材：陶粒、生化环、火山石。
* 去除对象：氨氮、亚硝酸盐、硝酸盐。

---

## 3. 综合净化模式（推荐）

### 3.1 "三段式生态净水系统"

1. **前端沉淀区**：去除大颗粒悬浮物
2. **植物梯级湿地区**：分级吸收氮磷
3. **循环曝气区**：维持系统稳定

### 3.2 观赏水体推荐组合

机械过滤 + 曝气 + 少量沉水植物

### 3.3 戏水水体推荐组合

砂滤 + 紫外线消毒 + 曝气（可选）

# **第三部分：量化参数篇**

## 1. 水深与生物抑制效果

### 1.1 生物抑制（沉水植物）常用水深

| 植物  | 最佳水深      | 来源依据       |
| --- | --------- | ---------- |
| 金鱼藻 | 40–80 cm  | 国内湖泊恢复工程参数 |
| 苦草  | 20–60 cm  | 水生植被重建指南   |
| 狐尾藻 | 30–100 cm | 生态护岸技术手册   |

### 1.2 抑藻的关键：透明度 ≥ 水深的 1/2

若水深 80 cm，则透明度应 ≥ 40 cm 才能维持沉水植物光照。

---

## 2. 曝气量与总水量的关系

### 2.1 经验公式

**曝气需求（L/min） ≈ 水体体积（m³） × 0.5–1.2**

示例：100 m³ 水体 → 曝气量 50–120 L/min

### 2.2 不同水体类型建议

| 水体类型     | 曝气需求               |
| -------- | ------------------ |
| 纯观赏水体    | 0.3–0.6 L/min/ m³  |
| 有鱼的水体    | 0.6–1.0 L/min / m³ |
| 生态水体（抑藻） | 1.0–1.2 L/min / m³ |

---

## 3. 植物与动物投放密度

### 3.1 水生植物

| 类型   | 投放密度           | 适用说明              |
| ---- | -------------- | ----------------- |
| 沉水植物 | 2–4 kg/m² 覆盖面积 | 维持 30–50% 水下面积覆盖率 |
| 浮叶植物 | 3–6 株/m²       | 控制遮光率 < 50%       |
| 挺水植物 | 4–9 株/m²       | 湿地区或浅滩区           |

### 3.2 鱼类（鲢、鳙为主）

| 鱼类 | 投放量              | 说明     |
| -- | ---------------- | ------ |
| 鲢  | 10–20 尾 / 100 m³ | 控制藻类   |
| 鳙  | 5–10 尾 / 100 m³  | 滤食浮游动物 |

### 3.3 贝类

| 贝类   | 投放密度       | 说明          |
| ---- | ---------- | ----------- |
| 三角帆蚌 | 2–4 只 / m² | 滤食悬浮物，提高透明度 |
